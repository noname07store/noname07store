3Cara Upload Konten Tutorial (Video/Gambar)

Opsi 1: Upload ke Hosting (Jika ada cPanel)

1. Masuk dashboard admin (panduan di atas)

2. Klik menu "Upload Konten"

3. Isi form tutorial (judul, kategori, deskripsi)

4. Upload file video/gambar

5. Klik "Publish Tutorial"

Opsi 2: Gunakan Cloud Storage (Recommended)

YouTube: Upload video ke YouTube ‚Üí Embed link

Google Drive: Upload file ‚Üí Get shareable link

Cloudinary: Upload gambar gratis ‚Üí Copy URL

Imgur: Upload gambar cepat ‚Üí Get direct link

4Integrasi dengan Backend (PHP/Database)

Untuk fitur lengkap dengan database:

1. Install XAMPP (localhost) atau gunakan hosting dengan PHP

2. Buat database MySQL untuk menyimpan data tutorial

3. Buat file upload.php untuk handle upload

4. Buat file api.php untuk CRUD operations

5. Ubah function JavaScript untuk connect ke backend

Struktur Database (MySQL):

tutorials table: id, title, category, description, video_url, instructor, rating, created_at

users table: id, name, email, password, role, created_at
